# 🏆 시간감쇠 기반 신작 랭킹 알고리즘 — 프로덕션 운영 중

2019년에 설계한 신작 랭킹 알고리즘이 **현재까지 실서비스 '최신출시' 탭에서 운영 중**입니다. 이 페이지의 스크린샷은 재현 샘플이 아니라 공개된 앱의 실제 화면입니다.

## 문제

기존 인기·매출 랭킹은 누적 지표 기반이라 **신작이 진입할 수 없는 구조**였습니다. 신작은 노출이 없으니 지표가 쌓이지 않고, 지표가 없으니 노출되지 않는 악순환 — 신작 발굴이라는 스토어의 핵심 가치가 랭킹에서 작동하지 않았습니다.

## 설계

- **신선도 부스트** — 출시 후 경과 시간에 지수 감쇠를 적용해, 신작에게 초기 노출 기회를 주되 시간이 지나면 자연스럽게 실력(지표) 경쟁으로 전환
- **다중 지표 정규화 가중합** — 다운로드·매출 등 스케일이 다른 지표를 정규화 후 가중합. 분산 안정화 변환으로 소수 대작의 지표 독식 방지
- **지표별 관측창 분리** — 지표 성격에 따라 집계 기간을 다르게 설정
- **평가 기준 사전 정의** — "좋은 랭킹"의 기준을 시뮬레이션 전에 먼저 정의하고, 실데이터 4,700건으로 검증 후 개발 이관

## 결과 — 실서비스 화면

'최신출시' 탭이 이 알고리즘으로 정렬됩니다. 설계 후 수년이 지난 지금도 프로덕션에서 그대로 운영되고 있습니다.
