# 🔄 RT Party by Persona.ly

`#리타겟팅` `#리텐션` `#증분분석` `#퍼스트파티` `#딥링크`

> 📅 **2026.05.12** · 🏢 **Persona.ly** · 📍 조선팰리스 강남

## 🔑 Key Findings

> **지난 세미나에서 나온 리텐션의 중요성 & 증분 분석이 다시 한번 강조되었음.** 이번 세미나에서는 조금 더 깊고 실무에 적용할 수 있는 이야기들을 들을 수 있었음
>
> - **리타겟팅의 중요성**
>   - 게임 시장은 성숙되어 있기 때문에 더 이상 UA가 되지 않음: 모객 단가 상승, 오디언스 풀 한정
>   - 앞으로 질적 성장이 이루어질 것임
>     1. 증분 증량 강화
>     2. 프로덕트 개선: 리타겟팅은 CRM 적용이 제일 중요하다.
>     3. 집행 비용 확장: 오디언스 확장보다 퍼스트 파티 데이터 내재화, 초개인화
> - **세밀한 리텐션 타겟팅의 방식에 대해서 들을 수 있었음**
>   - 리타겟팅의 조건: 게임 초반에 RT를 오히려 더 짧게 잡고, 중장기로 가면 복귀 구간을 확대한다. 런칭 1년 미만은 타이틀별 편차가 있다. RPG의 경우 좀 짧게 가져간다 — 업데이트가 많이 거치면 다시 돌아오기 어렵기 때문이다. 그래서 7일보다도 더 짧게 가져가는 경우들이 있다. 그래도 효과가 없으면 휴식기를 가져가거나 신규에 집중한다. 캐쥬얼의 경우 7일로 설정 (14일, 21일도 테스트 중)
> - **리텐션 분석 시 고려되어야 하는 점**
>   - 증분 분석: 리텐션 마케팅의 가장 큰 어려움은 '이 유저가 알아서 복귀한건가 아님 광고해서 복귀한건가'임. 이 차이를 보는 게 증분 분석임
>   - 퍼스트 파티 데이터를 중점적으로 봐야 함 (MMP와 차이가 있기 때문)

## 💡 Lessons Learned

현장에서 남긴 적용 메모:

- **리텐션 기준 재정의.** 우리 회사에서는 '결제여부'만 보기 때문에 '해당 게임 접속 여부' 기준으로 리텐션 고려하면 좋을 것 같음.
- **증분 분석 본격 도입.** 우리도 증분 분석(마케팅 효과 측정)을 본격적으로 고려하면 좋을 것 같음.
- **RT 소재는 UA처럼.** 패널 공통 의견 — '리텐션용 소재야' 티내는 건 효과적이지 않았고, UA처럼 보이게 하는 게 나음.

<summary>15:30 워크숍 — RT in 2026: 이제 선택이 아닌 필수</summary>

리타게팅은 전체 예산의 30% 정도 쓰이고 있으며 빠르게 확산 중.

휴면 기간은 개발사마다 다름.

카니발라이제이션을 우려하여 DSP를 여러개 쓰지 않는 경우가 많은데, 그러면 오히려 더 중복되며 여러개를 써도 카니발라이제이션을 걱정하지 않아도 됨.

(iOS, CTV의 고려도 좋을 것임)

**리타게팅이 주목받는 이유**

- 피로감·짧아진 집중력
- UA 비용 상승(신규 유저 확보 비용)
- RT는 퍼포먼스 팀의 기본 전략
- 경쟁 심화 — 선제적 대응 필요
- iOS: ATT 이후 어렵다는 인식 → 확률론적 솔루션으로 가능성 확대

**측정 세팅 점검**

- 측정 난이도에 대한 피드백이 많음
- Inactivity window는 앱·유저 특성에 맞게 설정(7일 고정 아님)
  - *캔디크래시형*: 소액 과금·LTV 회수 느림 → 비활성 기간 길게
  - *RPG/코어*: 이벤트마다 복귀 패턴 → 장르에 맞게 길게 잡는 경우
  - *경쟁 치열·LTV 회수 빠른 장르*: 이탈도 빠름 → 윈도우 짧게
- Re-engagement: 이벤트·매출과 UA의 double attribution 이슈, 너무 이른 재참여 유도는 비용 이중화 위험

**DSP 기본 원리 101**

- 대형 DSP는 실시간 경매·학습을 통해 입찰·노출 결정
- 카니발라이제이션: 다수 요소가 맞아떨어져야 의미 있는 영향으로 해석
- 공급 경로 복잡성
- 내부 경매: 고가치 유저 노출을 위해 DSP 1개만 쓰면 기회가 줄어들 수 있음 — 멀티 DSP 논의
- 멀티 DSP 신화: 고가치 인벤토리/스크린 타임은 원래 비쌈(미국 CPI 예시로 $212까지 언급)

**iOS 리타게팅**

- ATT로 IDFA 매칭 어려움, 동의율은 낮고(약 20% 언급) 그 집단은 LTV도 상대적으로 낮은 편
- 확률론적 접근: MMP 측(핑거프린트·UA·IP 등), DSP 측 모델(예: DNA — iOS 모델·UA·IP) 등
- 데이터 흐름 정확도 예시(약 85%) — 100%는 아니나 법적 이슈 없다는 설명, 입찰은 확신도 기반
- RT 전용 크리에이티브만 쓰지 말고 UA 소재 활용 권장
- iOS RT는 필수에 가까움, 일본·유럽에서 더 두드러진다는 언급

**CTV RT 기본**

- CTV = 인터넷 연결 TV 광고(예: 구글TV·애플TV)
- 골든 모먼트: 즉시 구매 의도는 낮을 수 있으나 주목도·여유 높음
- 1인 1가구, IP 기반 RT가 많음(iOS와 유사한 맥락)
- 클릭 없음 → VT 어트리뷰션 중요, 새 포맷이라 검증·기대치 관리 필요
- RT와 궁합은 좋으나 한국은 초기라 효과 불확실할 수 있음 — 일본·미국은 시도 가치
- 사기 취약성 언급 — 현실적 기대치 필요

<summary>16:30 세션 — 증분성 우선 / 딥링킹 / RT 부정 트래픽 (퍼스널리 서지은)</summary>

증분으로 광고 효과를 측정해야 함: 리텐션 마케팅의 어려운 점은 오가닉으로 올 사람을 돈 주고 데려온 것이냐 아님 광고로 데려온 것이냐의 차이를 보는 것임.

(리텐션 유저는 이미 앱을 설치했고, 실제로 삭제를 잘 안하기 때문에) Deep Link를 통해 흥미를 가질 페이지로 이동시키는 게 효과적임.

RT의 부정 트래픽을 탐지할 수 있는 방법 소개.

**Incrementality (증분)**

- 광고 없이도 돌아왔을까? 결제·멤버십·이벤트가 있는 이커머스형은 자연 복귀 확률이 높을 수 있음 → 멋진 ROAS와 별개로 허수 ROAS 함정
- 탐지: 컨트롤 vs 테스트 그룹, 복귀·매출 차이 = 증분
- 고려사항: 그룹 규모(예: 80:20, 수천~1만5천 이상에서 유효해지기 쉬움), 실제 도달, 사전 균질성 검증, 고정 오디언스의 시간에 따른 효율 저하 등
- 복귀율·매출 증분, 코호트별 매출로 장기 기여 확인

**Deep Linking**

- 클릭 → 원하는 화면 직행, 퍼널 단축 = 전환율에 유리
- 휴면 기간별 앱 삭제율 데이터: 예) 7일 미접속 시에도 상당 비율은 앱 미삭제 → 타겟팅에 참고
- 효과: IPM 3~5배 사례(비결제 유저 RT, 광고 수익화 맥락), 최대 5배까지 본 사례 언급
- 기본: 앱 오픈 딥링크만으로도 의도 파악에 도움
- 고급: 미사용 쿠폰·아이템 랜딩, 할인 적용 화면 등
- CRM·온디맨드 연동(캠페인 안정화 후): 장바구니 이탈, 푸시 무반응 휴면 등
- 사례: 우마무스메 — 구매 페이지 딥링크로 동일 캠페인 대비 ROAS 약 20% 상승(의도-랜딩 매칭 중요)

**RT 부정 트래픽**

- 요즘 fraud는 MMP를 교묘히 피해 과잉 attribution을 노리는 그레이 영역까지 포함한다는 설명
- 임포스터 증후군: 실제 증분 없이 좋아 보이는 경우 → A/B, 비활성 윈도우·입찰 조건으로 검증
- 비활성 정책·화이트리스트 인플레이션: 파트너가 윈도우를 임의로 줄이는 사례(예: 1일→0일) — 당일 UA 유입도 RT에 걸림, 증분 거의 없음. MMP UA 모객 vs RT 모수 raw 교차 검증 권장
- VT/CT 비율: 건강하면 CT가 압도적이어야 하는 경우가 많음(딥링크). VT가 과다면 의심 — VT 비중 가이드로 10% 전후 적당, 많아야 30% 수준 언급
- 배너 의존: 다이내믹 리마인더형은 의미 있을 수 있으나 게이밍은 배너 과다 시 캠페인 검증 권장
- Video loop: 짧은 영상 반복 재생 후 클릭 유도 + 딥링크 즉시 오픈 → 브랜드 경험 악화·위험. SSP/SDK 필터·MMP 크리에이티브 리포트·이상 클릭율/성과 스크리닝

시장의 변화(성숙, UA 어려움-단가 높음)에 따라 리타겟팅은 필수가 되고 있음.

리텐션 기간은 RPG의 경우 좀 빠르게, 캐쥬얼은 7일+로 하여 복귀 대상 유저 선정을 하고 있음.

광고 소재의 경우 AI를 적극 활용중임. 유저별로 다른 내용을 보여주고, 가장 효과적인 것을 빠르게 찾을 수 있음.

리텐션 유저의 광고 소재로 '리텐션용 소재야' 티내는 건 효과적이지 않았음 (UA처럼 보이게 하는 게 나음).

**Q1. 리타겟팅 활동은 어느 정도 수준까지 진행되고 있나요? UA와 RT간의 예산 배분 기준은?**

- 넥슨: 장기 서비스에 RT 필수. UA:RT 비율은 ROAS 기반으로 동적 조정. 초반 타이틀은 최소 RT, 중장기 복귀 구간에서 확대. 런칭 1년 미만은 타이틀별 편차.
- 스페이스V: 설치~가입~승인~재결제 풀퍼널. RT 약 30%. DSP 멀티로 지면 확장. 한계효율·풀 감소 시 회수 속도와 함께 판단.
- 드림에이지: RT 중요, ROAS 회수 기준. IP 팬덤형은 신규보다 복귀 중심. MMORPG는 업데이트(신규 서버 vs 고레벨 콘텐츠)에 따라 예산 민감.
- 카카오게임즈: 게임 업종에서 RT 필수. 신규 모객 비용 급증. 오디언스 매칭률 관점. 이탈 기준 세그먼트, 매출 확장 시 상위 과금 패널 중심. 중장기 타이틀 상시 비중 예시: 신규 3~4 : 복귀 6~7, 엔진 업데이트 시 5:5까지.
- 모비릭스: 벽돌깨기 등 하이브리드·ROAS 관리. 안정적 오디언스. RT 약 30%.

**Q2. 리타게팅 시작을 결정하게 된 주요 고려 사항은?**

- 스페이스V: 풀퍼널에서 결제 완주.
- 드림에이지: IP 한정 타겟, 이탈 유저 복귀가 생태계상 중요.
- 모비릭스: 장기 UA 기반 후 2024년부터 RT 본격, 미국 중심 확장.

**Q3. 최근 1~2년 사이 운영 방식·전략에서 가장 크게 바뀐 부분은?**

- 넥슨: 모객 단가 상승, 오디언스 풀 한정 → 정교한 타겟·자동화·소재 다양화. 타겟/리타겟 구분보다 효율 중심. 초기엔 퍼스트파티 효율, 한계 후 크리에이티브로 브로드닝 — 소액 테스트로 학습비 리스크 관리.
- 드림에이지: AI/ML 매체 타겟 고도화, 이탈 전 맥락을 소재에 녹이는 시도.

**Q4. 첫 분기 동안 가장 인상적이거나 예상과 달랐던 점은?**

- 넥슨: 증분 분석 시 리타겟에 올가닉이 많이 섞임. 그럼에도 복귀 1인 LTV는 우수. 빅미디어보다 DSP 효율이 좋은 사례도.
- 스페이스V: 메시지 비효율 시 비용 낭비 체감. CPA보다 기여·LTV 관점으로 재정의.
- 카카오게임즈: 증분은 디테일하게 못해도 올가닉과 단순 비례하진 않음. 세그먼트 명확한 DSP가 빅미디어보다 나은 경우.

**Q5. BI/BA 팀, 데이터 분석가, 제품팀이 성과 측정에 적극 참여하고 있나요?**

- 카카오게임즈: MMP 대시보드 기초 + 퍼스트/서드파티 유저 단 분석. BI와 협업해 UA/광고 인지 코드로 생애주기 분석 확대 중. DAU 등 간접 기여도 관찰.
- 스페이스V: RT는 협업 필수. CRM과 연계. 푸시 클릭이 아닌 행동(탐색·상세) 기반 세그먼트.
- 넥슨: 증분 실험 대량(2024~), 국가별 올가닉·내부 BI 지표를 1차 지표화. UA 분석가 전원 참여.

**Q6. 처음 시작할 때 내부 저항·우려와 리더십 설득 방법은?**

- 모비릭스: 안정적 유입이 있어 시도.
